# Fullstack-App: Flask + Postgres + Nginx

Учебный проект по контейнеризации и автоматизации деплоя. 
Основная цель — отработка навыков CI/CD, сетевой изоляции и управления инфраструктурой.

## Архитектура
- **Frontend**: Static HTML + JS
- **Backend**: Python (Flask) + psycopg2
- **Database**: PostgreSQL 15
- **Reverse Proxy**: Nginx (маршрутизация `/api/` -> Backend, `/` -> Frontend)

## Технологический стек
- **Docker & Docker Compose**: Контейнеризация и оркестрация.
- **GitHub Actions**: CI/CD пайплайн (сборка, проверка конфигов, авто-деплой).
- **Self-hosted Runner**: Использование локальной машины как сервера деплоя.
- **Security**: Сетевая изоляция (отдельные сети для БД и фронта), работа с Secrets.

## CI/CD Pipeline
При каждом push в ветку main срабатывает GitHub Action:

- Lint & Build Check: Проверка сборки Docker-образов и валидация конфига Nginx.
- Auto-Deploy: Если тесты прошли, Runner на целевой машине автоматически подтягивает код и перезапускает контейнеры.

## Безопасность

- База данных вынесена в изолированную сеть backend_net и недоступна извне.
- Все секреты (пароли БД) передаются через переменные окружения и замаскированы в GitHub Secrets.
- Используется .dockerignore для предотвращения утечки конфигов в образы.
